║                       PDF Tools 使用说明                           ║
╚════════════════════════════════════╝

用途：拆分 PDF、预览选页、文件命名、生成校验表。


【文件】
────────────────────────────────────
选择或拖入 1 个 PDF
  按拆分规则处理。

一次选择多个文件
  只做批量重命名，不拆分。

勾选“拖入单文件时预览拆分”
  拖入 1 个 PDF 后，会自动打开预览窗口。


【输出】
────────────────────────────────────
输出路径为空
  默认输出到当前目录。

输出目录不存在
  程序会自动创建。

输出目录已有文件
  运行前会询问是否清空。
  是：清空后继续。
  否：不清空，直接继续。
  取消：停止本次处理。

勾选“输出 CSV 校验表”
  会生成“拆分结果.csv”。


【预览拆分】
────────────────────────────────────
勾选页面
  按两个一组配对，作为拆分范围。

示例：
  勾选：1、3、6、10
  输出：1-3、6-10

如果最后只剩 1 个勾选页
  会按单页输出。

未选中的页面
  不会输出。

点击“使用选择”
  会把勾选结果写入拆分规则。


【规则拆分】
────────────────────────────────────
输入 3
  每 3 页拆成一个文件。
  例：10 页 PDF => 3、3、3

输入 2、3、5
  依次拆成 2 页、3 页、5 页。
  注意：剩余页面不会自动输出。

输入 1、3、*
  按 1 页、3 页循环拆分。
  例：8 页 PDF => 1、3、1、3
  如果最后一轮不完整，会弹窗提示。

支持写法：
  1,3,*
  1，3，*
  1、3、*
  逐行输入


【文件命名】
────────────────────────────────────
不填写
  程序自动生成文件名。

只填左侧
  按左侧顺序依次命名。

左右都填
  按 左1、右1、左2、右2 交替命名。

注意：
  不要写 .pdf 后缀。
  名称数量必须等于输出文件数量。
  名称不能重复。
  不会覆盖已有文件。


【日志和校验】
────────────────────────────────────
右侧日志
  显示拆分过程、输出文件、错误和耗时。

CSV 校验表
  记录文件名、期望页数、实际页数和校验结果。

页数不一致
  程序会提示。


【常用速查】
────────────────────────────────────
每 3 页拆一个文件
  拆分规则输入：3

按 2、3、5 页拆分
  拆分规则输入：2、3、5

按 1、3 页循环拆分
  拆分规则输入：1、3、*

手动选择页码范围
  打开预览，勾选起止页，再点“使用选择”。

给输出文件命名
  在“文件命名”里填写名称。

生成校验表
  勾选“输出 CSV 校验表”。
